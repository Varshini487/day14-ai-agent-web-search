# 🤖 AI Agents with Web Search + Memory — Day 14

An advanced AI agent that searches the web in real time AND remembers conversation history using the ReAct (Reason + Act) pattern.

## ✨ Features
- 🔍 Real-time web search using DuckDuckGo
- 🧮 Built-in calculator tool
- 🧠 Conversation memory across messages
- 🔄 ReAct loop: Thought → Action → Observation → Answer

## 🛠️ Tech Stack
- Python · LangChain · OpenAI GPT-3.5 · DuckDuckGo Search

## ⚙️ Setup
```bash
pip install langchain langchain-openai duckduckgo-search langchain-community
```
